## ⚠️ 승인을 위해 반드시 확인해야 할 항목

### 1. 콘텐츠 접근성 (가장 중요)
현재 모든 페이지가 **로그인 필수**로 설정되어 있습니다.
구글 크롤러는 로그인을 할 수 없어 콘텐츠를 볼 수 없습니다.

**해결 방법**: 아래 페이지들을 비로그인 상태에서도 접근 가능하게 변경
- 홈 (`/`) — 랜딩 페이지로 공개
- 포켓몬 도감 (`/pokedex`) — 콘텐츠가 풍부하여 승인에 유리

### 2. 콘텐츠 충분성
- 최소 **20~30개 이상의 고유 포켓몬 정보 페이지** 또는 그에 준하는 콘텐츠
- 이 앱은 1025마리 포켓몬 도감을 보유 → 콘텐츠 양은 충분

### 3. 저작권 주의
- 포켓몬 이미지/이름은 Nintendo/Game Freak 소유
- `팬메이드` 임을 명확히 표시 (현재 footer에 표시 중 ✅)
- 상업적 이용 시 저작권 이슈 가능성 있음
- 애드센스 심사 시 팬사이트로 통과되는 경우도 있으나 거절될 수도 있음

### 4. 트래픽
- 신규 사이트는 일정 트래픽이 있어야 승인 가능성 높아짐
- 배포 후 SNS 공유 등으로 방문자 유입 필요
